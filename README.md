# personal-site

Personal website — [hameddaoud.pages.dev](https://hameddaoud.pages.dev).

Built with [Astro](https://astro.build) on the [AstroPaper](https://github.com/satnaing/astro-paper) theme. Deployed on Cloudflare Pages.

## Local dev

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # static build into dist/
```

Node >= 22.12.0.
